Evented is a Chrome Extension that helps you convert highlighted text or manual input into fully structured Google Calendar events using AI (OpenAI). It reads event-like text (e.g. "Dinner at 7pm at John's") and schedules it automatically to your Google Calendar.

How it works

  • You can either highlight text on a webpage or manually type/paste it into the built-in textbox.
  • Click "Create Event" to send the input to OpenAI and generate an event.
  • If youâ€™ve signed into Google, the event will be added directly to your Calendar.

Using Your Own OpenAI API Key

For privacy and to avoid sharing keys, you must use and pay for your own OpenAI API key.
1. Visit OpenAI API Keys to create a key.
2. Paste your key into the Settings section of the extension popup.
3. You can choose from the two supported GPT model (e.g. gpt-3.5-turbo, gpt-4).

You are responsible for any costs incurred. See full pricing here: OpenAI Pricing.

Evented never stores your API key or data externally. All processing happens locally in the extension, and event creation only occurs when you explicitly allow access to your Google Calendar.
