Privacy Policy

Effective Date: Aug 2, 2025

Evented does not collect, store, or transmit personal data to any external server controlled by us.

  • Text you highlight or input stays local unless you explicitly submit it to the OpenAI API.
  • API keys and user preferences are securely stored using Chrome's storage.local API. This data never leaves your browser.
  • When you authorize access to Google Calendar, only the permissions required to create events are used. No other calendar data is read or stored.
  • Your Google account details are never stored or shared by Evented.

This extension uses OpenAI’s API for natural language processing. Submitted content is sent to OpenAI for processing and is subject to OpenAI’s terms of service.

We do not use cookies, analytics tools, or third-party trackers.
